# Florence Denys

Artiste travaillant la porcelaine.

À travers la porcelaine, j'explore des formes qui résistent à l'identification et ouvrent un espace de projection vers un ailleurs impossible à nommer.

## Pages

- **Accueil** - Page d'introduction
- **Œuvres** - Galerie de travaux
- **À propos** - Bio et pratique artistique
- **Contact** - Informations et formulaire de contact

## Comment utiliser

1. Clonez le repository
2. Remplacez les placeholders par votre contenu :
   - Images (`placeholder.jpg`)
   - Informations de contact
   - Texte bio et descriptions

3. Publiez sur GitHub Pages ou un serveur web
